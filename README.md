# cool-website1

Luxury-themed Next.js 15 app using TypeScript, Tailwind CSS, shadcn-style UI components, React Hook Form + Zod, Framer Motion, and middleware-based route protection.

## Run locally

1. Install dependencies:
   `npm install`
2. Start dev server:
   `npm run dev`

## Routes

- `/` Home (public)
- `/signup` Sign up
- `/login` Login
- `/dashboard` Protected
- `/profile` Protected
